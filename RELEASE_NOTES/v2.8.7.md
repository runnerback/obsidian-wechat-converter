## 更新内容

### 🚀 重大更新
- **上线官方中转代理服务**：全新上线高稳定、免自建的官方中转代理服务（29元/年），配合专属 Token 与设备防超额绑定机制，解决写作者多地办公/移动热点带来的微信公众平台 IP 白名单频繁漂移、同步失败的痛点。

### ✨ 功能优化
- **中转鉴权报错友好化**：对于代理 Token 填错、服务到期或设备超载（403/401错误），直接显示中转服务器返回的具体原因（如 “Token 无效”），并自动熔断网络重试（零延迟响应报错）。
- **同步弹窗排障引导**：代理鉴权失败时，同步错误弹窗自动切换为专属的代理配置排查指引，并过滤隐藏“取消关联并新建草稿”等无关联干扰按钮。
- **自动重传规避封面过期**：为内存封面缓存引入上传时间戳校验。当在内存中已保存超过 2.5 天，下次同步时自动废弃缓存并重传，彻底解决 Obsidian 客户端长时间挂机（不关闭侧边栏预览）导致的封面图素材过期报错。
- **错误代码 40007 友好化**：同步微信草稿或上传封面时，若触发微信侧 40007（媒体 ID 失效）异常，自动将其转换为中文白话排障说明，引导用户一键点击弹窗下方的“取消关联并新建草稿”进行自动恢复。
