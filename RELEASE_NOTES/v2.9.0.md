---

## 更新内容

### 🚀 重大更新
- 新增飞书云文档发布能力：可将当前 Obsidian 笔记直接发布到配置好的飞书文件夹，并支持后续基于已绑定文档进行覆盖更新。
- 飞书同步支持文档重绑：当飞书侧文档被移动、删除后重建，或本地缓存 token 失效时，可通过 docx 链接重新绑定当前笔记。
- 飞书媒体同步增强：支持本地路径、相对路径、WikiLink、远程图床图片与 GIF 动图上传，GIF 在飞书文档内可保持播放。

### ✨ 功能优化
- 补齐飞书语法映射与发布体验，包括调用次数统计、任务列表、Callout、数学公式以及 Mermaid 处理策略。
- 优化发布弹窗与插件设置页的飞书体验，包括更清晰的目标平台结构、滚动区域、状态提示与设置说明层级。
- README 中补充飞书云文档与多平台发布说明，并更新对应截图，便于新用户快速理解工作流。

### 🐛 问题修复
- 修复飞书智能覆盖更新中的多类异常场景，包括旧 token 失效、删除块失败、插入块 schema mismatch、远程图片 CORS 读取失败等问题。
- 修复若干 Obsidian scan / review warning，包括 unsafe call、typed warning、Kroki base64 与扫描门禁相关问题。
- 改善浏览器插件 bridge 的错误分类与提示，减少多平台发布中的误导性报错。

### ✅ 质量保障
- 发布前通过 `npm run scan:guard` 与 `npm run review:guard`。
- 版本号升级到 `2.9.0`，保持 `minAppVersion` 为 `1.4.0`。
