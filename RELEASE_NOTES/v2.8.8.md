## 更新内容

### 🚀 重大更新
- **排除打包静态库（修复社区扫描）**：将 `lib/` 目录下的大体积静态库文件（如 `markdown-it.min.js`, `highlight.min.js`, `mathjax-plugin.js` 等）从 Git 追踪中剔除，移至本地构建阶段由 `esbuild` 自动动态编译打包。解决了因为 Obsidian 官方测试扫描系统遍历未压缩静态依赖时耗时过长导致的审查崩溃/超时（Internal Error）痛点。

### ✨ 功能优化
- **待办列表删除线渲染**：优化待办事项（Checklist）的渲染逻辑，支持在微信排版中将已完成的待办事项渲染上删除线样式（Strikethrough），呈现更符合主流文档直觉的待办样式。

### 🐛 问题修复
- **ESLint 本地与 CI 扫描配置过滤**：优化了项目根目录的 ESLint 扫描规则配置，防止本地和 GitHub Actions 扫描时对动态构建产物产生无用警告。
