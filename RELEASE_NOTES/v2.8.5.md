## 更新内容

### 🐛 问题修复
- 修复「网格文档」主题粘贴或同步到微信公众号后，在深色模式下正文颜色过暗、长时间停留后可读性下降的问题。
- 保留浅色模式下原有的白色网格文档视觉，不再为了深色模式牺牲主题特色。
- 调整列表和引用等内容的颜色继承方式，让微信自身的深色模式转换能够稳定生效。

### ✨ 功能优化
- 移除 Obsidian 1.4.5 仅部分支持的 CSS Mask 用法，Pro 标识改用兼容性更好的字符图标。
- 调整内嵌 Markdown 运行时的国际化域名标准化实现，保留裸域名链接识别和 Punycode 转换能力，同时避免触发隐藏网络端点的静态扫描提示。
- 更新依赖锁定结果，`@xmldom/xmldom` 统一使用安全版本 `0.9.10`。
- 中文 README 恢复为纯中文介绍；独立英文文档继续保留在 `README.en.md`。

### 🧪 测试与质量
- 新增网格文档深色模式、非网格主题隔离、列表颜色继承和微信输出兼容性测试。
- 新增内嵌 Markdown 裸域名链接与国际化域名转换回归测试。
- 完整测试、构建及发布包校验均通过。
