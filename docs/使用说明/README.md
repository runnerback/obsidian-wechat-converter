## 设置面板速览

打开：Obsidian 设置 → 第三方插件 → 微信发布助手（或转换器面板右上角齿轮）。

「微信」标签页从上到下的主要区块：

1. **公众号账号** —— AppID / AppSecret / 封面摘要默认值等（多账号）
2. **预览 / 水印 / 排版** —— 手机框、图片说明、留白等
3. **AI Provider** —— AI 能力的共享凭证（当前 DeepSeek），详见 [AI Provider 设置](./AI-Provider设置.md)
   - 折叠「AI 编排」—— 一键 AI 排版
   - 折叠「标题 AI 润色」—— 一键优化标题
4. **高级设置 → API 代理地址** —— 见 [API 代理设置](./API代理设置.md)
