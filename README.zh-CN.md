# Content Studio

> 版本 3.9.40 · 更新 2026-07-14 · [English](./README.md)

把 Obsidian 笔记变成可直接发布的内容：**微信公众号**文章、**小红书**图文卡片、**飞书**云文档——全部在 Obsidian 内实时预览。

## 功能

- **公众号文章** —— Markdown 实时预览为公众号排版效果（代码块、引用、本地图片、GIF、公式与图表全部适配），一键复制进公众号编辑器，或直接保存到账号草稿箱。
- **小红书图文卡片** —— 预览切到「小红书」模式：每个标题生成一张图卡，可选多套主题（含 iOS 备忘录风格），导出 PNG 或推送到小红书草稿箱。
- **飞书云文档** —— 同一篇笔记同步到飞书云文档。
- **AI 编排与标题润色** —— 可选的 AI 排版与标题建议。

## 快速上手

1. 点击侧边栏 **Content Studio** 图标（或运行命令「打开Content Studio」）打开预览面板；
2. 编辑笔记，面板实时渲染，双向同步滚动；
3. 顶栏切换平台（公众号 / 小红书）、调整样式，通过「**发布与分发**」发布。

## 配置

- **公众号**：在设置中填入 AppID/Secret。微信 API 有 IP 白名单限制，请求需经你自己部署的代理转发，见 [`PROXY_GUIDE.md`](./PROXY_GUIDE.md)。
- **飞书**：在设置中填入飞书应用凭证。
- **AI 功能**（可选）：在设置中配置 API Key。
- **小红书草稿发布**（可选）：配对配套浏览器扩展 **多栖 Crosspost**（设置 → 其他平台）。仅导出 PNG 图卡无需扩展。

## 手动安装

从最新 Release 下载 `main.js` / `manifest.json` / `styles.css` 放入 `.obsidian/plugins/content-studio/`，然后启用插件。
