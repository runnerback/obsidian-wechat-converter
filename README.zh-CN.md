# Content Studio

> 版本 3.9.31 · 更新 2026-07-13 · [English](./README.md)

一款 Obsidian 内容排版与发布插件。把 Markdown 笔记排版预览成**微信公众号**文章和**小红书**图文卡片，同步到**飞书**云文档，并通过配套的浏览器端发布助手 **多栖 Crosspost** 推送到平台草稿箱（当前为小红书）。

## 功能

- **公众号预览与导出** —— 将 Markdown 转成符合公众号生态的 HTML（代码块、引用、列表、本地图片、GIF、公式与图表），一键复制到公众号编辑器或保存为草稿。
- **小红书图文卡片** —— 预览区切到「小红书」模式，渲染图卡并发布图文笔记到小红书草稿箱。
- **飞书云文档** —— 同一篇笔记同步到飞书云文档。
- **配套发布助手** —— **多栖 Crosspost** 是本插件的浏览器端发布助手（非独立工具）：接收本插件推送的内容，用你已有的浏览器登录态保存为平台草稿。当前对接小红书，更多平台规划中。
- **AI 编排与标题润色** —— 可选的 AI 排版与标题建议。

## 安装（本地构建）

```bash
npm install
bash dev-install.sh   # 构建并部署到 Obsidian 插件目录
```

或手动执行 `npm run build`，把 `main.js` / `manifest.json` / `styles.css` 放进 `.obsidian/plugins/wechat-converter/`。

## 配套浏览器助手（多栖 Crosspost）

发布到平台草稿箱由 **多栖 Crosspost** 完成——它是本插件的浏览器端发布助手，**不单独使用**：接收 Content Studio 经本地桥接推来的内容，写入平台草稿（当前为小红书，更多平台规划中）：

- **下载**：目前仅支持从 GitHub 下载（Chrome 应用商店待上架）。
- **安装**：Chrome → 扩展程序 → 开启开发者模式 → 加载已解压的扩展程序 → 选择扩展的 `dist` 目录。
- **配对**：打开扩展弹窗 → 设置 → 复制「连接令牌」→ 填到本插件设置的「其他平台」标签页。
