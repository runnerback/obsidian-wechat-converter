# 微信公众号 API 官方中转代理服务配置指引

感谢您订阅本插件的官方中转代理服务。为了能够正常将文章同步至您的微信公众号草稿箱，请按照以下指引完成配置。

---

## 1. 插件端配置

打开 Obsidian 插件设置，找到 **微信官方/草稿箱同步** 选项下的 **API 代理地址**，将您专属的中转服务地址填入输入框中：

### 🔗 代理地址格式
```text
https://wechat-api.xiaoweibox.top/proxy?token=wp_user_YOUR_TOKEN
```
> [!IMPORTANT]
> - 请将上述链接中的 `wp_user_YOUR_TOKEN` 替换为您购买服务后获得的**专属 Token**（通常以 `wp_user_` 开头）。
> - 请确保整个链接使用 `https://` 开头，且没有任何多余的空格。

---

## 2. 微信开发者平台配置（配置 IP 白名单）

微信公众号 API 接口为了安全起见，要求发起请求的服务器 IP 必须在公众号的 **IP 白名单**中。您需要将本中转服务的服务器 IP 添加到白名单。

### 📌 中转服务服务器 IP
```text
43.156.58.84
```

### 🛠 配置步骤：
1. 访问并登录 [微信开发者平台](https://developers.weixin.qq.com/platform)（或微信公众平台后台）。
2. 在左侧菜单中找到 **设置与开发** -> **基本配置**。
3. 找到 **IP 白名单** 栏目，点击 **查看/修改**（若未开启，请点击开启）。
4. 将中转服务服务器 IP `43.156.58.84` 粘贴到输入框中。
   - *提示：如果已存在其他 IP，请换行或用英文逗号分隔添加。*
5. 保存并进行管理员扫码确认，以使白名单配置生效。

---

## 3. 设备与安全说明
- **多设备限制**：为了保障服务不被滥用，单个 Token 默认支持在您个人的 **3 台不同设备**（例如：主力电脑、公司电脑、备用电脑）上使用。
- **设备过期机制**：中转服务会记录最近 15 天内发起过请求的设备 ID，如果您更换了新电脑或重装了系统，超过 15 天未使用的旧设备记录会自动清除，无需担心设备超限问题。
- **隐私与安全**：您的公众号敏感凭证 `AppSecret` 仅在本地 Obsidian 插件和微信官方服务器之间通过加密通道中转，中转服务器**不存储任何您的 AppSecret 或账号信息**。
